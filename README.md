# Сетевой чат на языке C (TCP-сокеты)

Многопользовательский чат-сервер с поддержкой личных и общих сообщений. Реализован на языке C с использованием TCP-сокетов и многопоточности (pthread).

## Возможности

- ✅ Поддержка множества одновременных подключений (каждый клиент в отдельном потоке)
- ✅ Выбор псевдонима при подключении
- ✅ Общие сообщения (рассылка всем, кроме отправителя)
- ✅ Личные сообщения (отправка конкретному пользователю)
- ✅ Просмотр списка активных пользователей
- ✅ Устойчивость сервера к отключениям клиентов
- ✅ Логирование подключений и отключений
- ✅ Корректное завершение через `/exit`
- ✅ Обработка сигнала SIGINT (Ctrl+C) на сервере
